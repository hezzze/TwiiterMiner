TwiiterMiner
============

Twitter purpose mining

## extracted_data.json

An array of arrays, each array contains the values of the wanted fields strictly ordered as the following:

- created_at
- id
- text
- in_reply_to_status_id
- in_reply_to_user_id
- geo
- place
- retweet_count
- retweeted
- user/id
- user/name
- user/screen_name
- location
